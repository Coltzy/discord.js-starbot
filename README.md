## Installation

- [Fork](https://github.com/Coltzy/discord.js-starbot/fork) the repository.

```
$ git clone https://github.com/your-username/discord.js-starbot.git
$ cd discord.js-starbot
$ yarn install
```

- Add the follwing enviromental variables

```
MONGO_URI=***
DISCORD_TOKEN=***
CLIENT_ID=***
```
